# 低成本短信转发器

>
本项目是低成本短信转发器的一个修改版，原方案请前往 [chenxuuu/sms_forwarding](https://github.com/chenxuuu/sms_forwarding) 。

本项目为短信转发器增加转发到企业微信应用。

需要在`Arduino IDE`中增加安装 **ArduinoJson** 库。
